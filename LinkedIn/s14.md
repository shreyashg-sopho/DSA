### Optimal Code

```java
class Solution {
    public int maxDepth(TreeNode root) {
        return heightOfTree( root);
    }
    
    
    private int heightOfTree(TreeNode root)
    {
        if(root == null) return 0;
        return 1 + Math.max(heightOfTree(root.left) ,heightOfTree(root.right) );
    }
}
```
